# 🌉 ツール間連携 (Interop)

異なる Ansys 製品間、あるいは外部データと Ansys のやり取りを効率化するテクニックとスクリプトです。

## コンテンツ

- **[STEP Import Tricks](/interop/step-import-trick/color_named_selection)**:
  - **[Color to Named Selection](/interop/step-import-trick/color_named_selection)**: STEP ファイルの色情報を Mechanical の Named Selection に自動変換する手法。
- **[Workbench to Mechanical](/interop/wb-to-mech/pass_parameters)**:
  - **[Pass Parameters](/interop/wb-to-mech/pass_parameters)**: Workbench のパラメータを Mechanical スクリプト内で活用する方法。

## 提供価値

解析プロセスのボトルネックとなりやすい「ツール間のデータ受け渡し」を自動化し、ヒューマンエラーを削減します。
