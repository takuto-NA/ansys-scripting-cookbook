# 🏷️ Named Selection (Groups)

SpaceClaim における Named Selection (Group) の作成と管理に関するスクリプトです。

## 解説

SpaceClaim では、Named Selection は **Groups** パネルで管理されます。
スクリプトで作成したグループは、Mechanical にインポートした際に自動的に Named Selection として認識されます。

## スニペット例

具体的な作成例は、[Interop: Color to Named Selection](../../interop/step-import-trick/color_named_selection.md) 内の SpaceClaim スクリプトを参照してください。
